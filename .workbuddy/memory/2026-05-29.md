# 2026-05-29 工作记录

## 阅读追踪器移动端 App 设计

基于现有阅读追踪器项目，在 Ardot 画布上完成了移动端 App 的 4 个核心屏幕设计：

1. **书架主页** — 书籍列表、搜索、筛选、底部导航
2. **年度统计** — 数据统计卡片、月度阅读量柱状图
3. **书籍详情** — 封面、评分、读后感、标签
4. **添加书籍** — 搜索表单、状态选择、EPUB 导入

### 设计风格
- **E-Ink / Paper** 纸质感风格
- **配色**: 暖奶油底 #FFFBEB + 琥珀色强调 #D97706 + 深棕文字 #0F172A
- **字体**: Cormorant Garamond (标题) + Libre Baskerville (正文)
- **特点**: 0px 圆角、无阴影、极简 UI、阅读优先

### 关于 DMG 打包
用户提到后续需要打包成 .dmg 文件。当前项目基于 Next.js + React + TypeScript，后续可考虑使用 **Tauri** 或 **Electron** 将现有 Web 应用封装为桌面应用并打包为 macOS DMG。设计稿可直接复用为桌面端界面。
